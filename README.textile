h1. Ads for phpWebSite

"Ads":http://github.com/meisteg/ads is a module for the "phpWebSite":http://phpwebsite.appstate.edu CMS that lets you add advertising to your site.  It supports banner and text ads, and supports multiple ads per page.  Ad statistics such as views and hits are recorded.

h2. Getting Started

* "Installation Instructions":http://wiki.github.com/meisteg/ads/install for this module.
* "Submit an issue":http://github.com/meisteg/ads/issues if you have questions or to report a problem.
